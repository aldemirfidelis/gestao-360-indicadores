Como usar

Abra este arquivo no navegador para ver os fluxogramas renderizados. Use Imprimir / Salvar PDF para gerar o PDF de distribuição, ou baixe SVG de cada diagrama. A implementação no app altera apenas navigation.ts — rotas existentes são mantidas.

FSMS + Patrimonial Módulos próprios (como Gestão de Prêmio)
Lançamentos + Relatórios Expandidos com entradas operacionais e auditoria
